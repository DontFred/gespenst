# Table

The Table components are used to render text content in Gespenst. It is a
wrapper around native HTML Table elements.

## Import

```tsx
import { Table, TBody, TD, TH, THead, TR } from "gespenst/text";
```

## API

**Special:** Every component is just a wrapper around the native HTML element.
So all props are the same for each component to there counter part. e. g.
`Table` has every attribute from `HTMLTableElement`

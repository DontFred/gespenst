# Text

The Text component is used to render text content in Gespenst. It is a wrapper
around native HTML elements.

## Import

```tsx
import { Text } from "gespenst/text";
```

## Usage

## API

<Text large className="mt-2">{`TextProps.*`}</Text>

**Special:** Only one format can be toggled at a time.

**Special:** Formats manage the rendered element and therefor the HTML props. \
e.g. If TextProps.**heading** is `true`, the text renders as a heading element. So
it has additionally every prop from the `HTMLHeadingElement`.

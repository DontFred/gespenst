# Toggle

The Toggle component is used to render fully accessible Toggle. It is wrapper
around the react-aria-component
[Switch](https://react-spectrum.adobe.com/react-aria/Switch.html).

## Import

```tsx
import { Toggle } from "gespenst/toggle";
```

## API

**Special:** Toggle is just a styled form of the react-aria-components toggle.
So all props are the same as in the
[Switch](https://react-spectrum.adobe.com/react-aria/Switch.html#props) without
considering the additional props.

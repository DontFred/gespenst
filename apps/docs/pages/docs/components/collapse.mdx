# Collapse

The Collapse component display a list of high-level options that can
expand/collapse to reveal more information.

## Import

```tsx
import {
  Collapse,
  CollapseItem,
  CollapseTrigger,
  CollapseContent,
} from "@gespenst/ui/collapse";
```

# Loading Spinner

The Spinner component is used to render a spinner that indicates a loading
state. It is a wrapper around native `HTMLDivElement`.

## Import

```tsx
import { Spinner } from "gespenst/spinner";
```

## Usage

<Tabs items={["Preview", "Code"]}>

    <Card className="grid">

  <Tabs.Tab>
```tsx

  return (
    <Spinner/>
  );
}
 ```
  </Tabs.Tab>
</Tabs>

## API

**Special:** The Spinner component does not have any custom props. It only uses
the default `HTMLDivElement` props.

# Button

The Button component is used to render fully accessible button. It is wrapper
around the react-aria-component
[Button](https://react-spectrum.adobe.com/react-aria/Button.html).

## Import

```tsx
import { Button } from "gespenst/button";
```

## API

**Special:** Button is just a wrapper around the react-aria-components button.
So all props are the same as in the
[Button](https://react-spectrum.adobe.com/react-aria/Button.html#props) without
considering the additional props.

**Special:** Only `size="xsmall"` can be used with `svgOnly={true}`.

**Special:** When`svgOnly={true}` is used `prefix` and `suffix` are not allowed.

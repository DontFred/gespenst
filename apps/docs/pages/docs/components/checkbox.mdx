# Checkbox

The Checkbox components are used to render fully accessible Checkbox and
CheckboxGroup. They are wrapper around the react-aria-component
[Checkbox](https://react-spectrum.adobe.com/react-aria/Checkbox.html).

## Import

```tsx
import { Checkbox, CheckboxGroup } from "gespenst/loading-spinner";
```

## API

**Special:** Checkbox components is just a styled form of the
react-aria-components Checkboxs. So all props are the same as in the
[Checkbox](https://react-spectrum.adobe.com/react-aria/Checkbox.html#props) and
[CheckboxGroup](https://react-spectrum.adobe.com/react-aria/CheckboxGroup.html#props).

# Skeleton

The Skeleton component is used to render a Skeleton that indicates a loading
state. It is a wrapper around native `HTMLSpanElement`.

## Import

```tsx
import { Skeleton } from "gespenst/skeleton";
```

## Usage

## API

**Special:** The Skeleton component does not have any custom props. It only uses
the default `HTMLSpanElement` props.

# Error

The Error component is used to render an error message. It is a wrapper around
`Icon` component.

## Import

```tsx
import { Error } from "gespenst/error";
```

## Usage

<Tabs items={["Preview", "Code"]}>

      <Error containerClassName="place-self-center">
        {`Something went wrong!`}
      </Error>
    </Card>
  </Tabs.Tab>

## API

default `IconProps` and the following custom props.

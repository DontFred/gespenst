# Modal

The Popover components are used to render fully accessible modal. The main
components are wrapper around the react-aria-component
[Modal](https://react-spectrum.adobe.com/react-aria/Modal.html). The other
components are wrapper around the `HTMLDivElement`.

## Import

```tsx
import {
  Modal,
  ModalActions,
  ModalBody,
  ModalHeader,
  ModalInset,
  ModalSubtitle,
  ModalTitle,
  ModalTrigger,
} from "gespenst/modal";
```

## Use Close Button

<Tabs items={["Preview", "Code"]}>
  <Tabs.Tab>
    <Card className="grid">
      <CloseExample/>
    </Card>
  </Tabs.Tab>
  <Tabs.Tab>
```tsx
import {
  Modal,
  ModalActions,
  ModalBody,
  ModalHeader,
  ModalSubtitle,
  ModalTitle,
  ModalTrigger,
} from "gespenst/modal";
import { Fragment } from "react";

export default function App() {
  return (
    <ModalTrigger>
      <Button>Open Modal</Button>
      <Modal>
        {({ close }) => (
          <Fragment>
            <ModalBody>
              <ModalHeader>
                <ModalTitle>Modal Title</ModalTitle>
                <ModalSubtitle>Modal Subtitle</ModalSubtitle>
              </ModalHeader>
              <Text paragraph>
                Lorem ipsum dolor sit amet, consectetur adipiscing elit.
              </Text>
            </ModalBody>
            <ModalActions>
              <Button
                onPress={() => {
                  close();
                }}
                variant={"secondary"}
              >
                Close
              </Button>
            </ModalActions>
          </Fragment>
        )}
      </Modal>
    </ModalTrigger>
  );
}
 ```
  </Tabs.Tab>
</Tabs>

## API

**Special:** The Modal components are just a styled form of the
react-aria-components Modals. So most of the props are the same as in the
[Modal](https://react-spectrum.adobe.com/react-aria/Modal.html#props). But the
Content components are wrapper around `HTMLDivElement`.

- Modal is a styled react-aria-components `Modal`
- ModalActions is a styled `HTMLDivElement`
- ModalBody is a styled `HTMLDivElement`
- ModalHeader is a styled `HTMLDivElement`
- ModalInset zis a styled `HTMLDivElement`
- ModalSubtitle is a styled `HTMLDivElement`
- ModalTitle is a styled react-aria-components `Heading`
- ModalTrigger is a re-export of `DialogTrigger`

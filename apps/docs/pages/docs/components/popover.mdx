# Popover

The Popover components are used to render fully accessible popover. It is
wrapper around the react-aria-component
[Popover](https://react-spectrum.adobe.com/react-aria/Popover.html).

## Import

```tsx
import { Popover, PopoverTrigger } from "gespenst/popover";
```

export default function App() {
  return (
    <PopoverTrigger>

## API

**Special:** Popover components is just a styled form of the
react-aria-components Popovers. So all props are the same as in the
[Popover](https://react-spectrum.adobe.com/react-aria/Popover.html#props) and
[DialogTrigger](https://react-spectrum.adobe.com/react-aria/Dialog.html#dialogtrigger)
(as PopoverTrigger).

# Badge

The Badge component is used to render a simple badge with a text. It is a
wrapper around native `HTMLSpanElement`.

## Import

```tsx
import { Badge } from "gespenst/badge";
```

## Usage

## API

**Special:** If using the `BadgeProps.children` attribute, the Badge will be
fixed to the right corner of the child.

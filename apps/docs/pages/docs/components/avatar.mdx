# Avatar

The Avatar components are used to render an fully error handled avatar and
avatar group. They are custom made with useState for error handling and can be
used with the
[next/image](https://nextjs.org/docs/pages/api-reference/components/image)
component, if NextJs is used.

## Import

```tsx
import { Avatar, AvatarGroup } from "gespenst/button";
```

## API

**Special:** The Avatar component is using the `HTMLSpanElement`. So you can use
the native attributes of it.

**Special:** If you are using NextJS you can use the NextImage attribute to add
the Image component.

**Special:** The AvatarGroup component is using the `HTMLDivElement`. So you can
use the native attributes of it.
